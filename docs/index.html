    1. HOME

    hadv-book を始めましょう #

    ここは hadv-book のトップページ（content/_index.md）です。
    ここを書き換えて、プロジェクト・サイトのホームページとします。

    hadv-book の紹介 #

    かんたん環境構築

    hadv-book は、簡単に開始できます。
    Windows に WSL2 を使用した Docker Desktop と VScode と Git さえあれば他に何もいりません。
    いまご覧になっているページをすぐに手元に再現できます。
    使い方は Usage にまとめました！

    AsciiDoc 対応

    めんどうな Asciidoctor のインストールも不要です。
    Asciidoctor の公式 Docker イメージを使ってビルドをしますので、Windows 環境も汚しません。
    最新版への対応も簡単です、Docker イメージのバージョンを上げるだけ！

    Theme について

    hadv-book は、Hugo Book Theme （commit）をカスタマイズして使用しています。
    もちろん Book 以外の Theme にも変更は可能です。サポートはしませんが、ご自由にどうぞ！
    （その場合、hadv-bookではなくなってしまいますが …）

    高速ビルド

    Hugo は数ある静的サイトジェネレーターの中でも、Web サイトのビルドが一番はやい！
    記事を書くたびに蓄積されていくストレスもありません。コンテンツの数が増えても安心です。

    初期サイドメニューについて #

    Usage

    Usage（content/docs/usage）には、

    • hadv-book の開始方法（→ Getting Started）
    • テンプレート集（→ Templates）

    などがあります。

    Example

    Example（content/docs/example）には、

    • Markdown（→ Markdown）
    • AsciiDoc（→ AsciiDoc）

    … 形式の実例ページを用意しました。
    ドキュメント作成時の参考になります。

    Blog

    Blog（content/posts）のサンプルを表示します。
    サンプルを削除して新たに記事を追加すれば、すぐに使い始められます。
